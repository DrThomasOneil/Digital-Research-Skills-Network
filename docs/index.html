  • Workshop schedules and announcements.
  • Step-by-step instructions for workshop activities.

Workshop material will aim to use publicly available data or mock created data. Therefore, you will not need to rely on uploaded data from the demonstrators/creators. Instead, a pre-workshop set-ups will include instructions for where to download and prep relevant data.

Bookmark this site for quick access to scripts and updates as workshops progress.

Using this site

This site is organised into several sections to help you quickly find what you need:

  • Getting Started in R: A beginner-friendly guide to help you set up R and RStudio, along with key concepts and tips for coding.
  • Announcements: Updates on upcoming workshops, deadlines, and important notices.
  • Workshops: Detailed instructions, scripts, and examples for each workshop session.
  • Extra Resources: Links to helpful tutorials, data sets, and cheat sheets.
  • Contacts: Acknowledge who contributed to these workshops! :)
